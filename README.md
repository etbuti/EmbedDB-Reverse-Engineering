# EmbedDB Reverse Engineering Toolkit

An open-source toolkit for understanding, analyzing, and extracting resources from legacy EmbedDB containers.

Current capabilities:

- Resource discovery
- Record layout analysis
- Binary structure inspection
- HTML / image / audio recovery
- Evidence-driven reverse engineering workflow

Originally developed as the backend toolkit for the Open English Learning Archive project.
